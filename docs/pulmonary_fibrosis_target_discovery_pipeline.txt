1. OVERVIEW AND RATIONALE
================================================================================

GOAL: Identify receptors on alveolar epithelial progenitors whose activation:
  - Maintains/expands functional AT2 cells (alveolar stem cells)
  - Drives productive AT2 -> AT1 differentiation (alveolar regeneration)
  - Prevents arrest in the pathological transitional state
  - Suppresses fibroblast-to-myofibroblast conversion (indirectly, via epithelium)

BIOLOGICAL CONTEXT:
  - In IPF/PF-ILD, repeated AT2 injury + failed regeneration drives fibrosis
  - FGF10 from the stromal niche (alveolar fibroblasts, ASMC, myofibroblasts)
    signals through FGFR2b on epithelium to promote AT2 survival and repair
  - FGF10/FGFR2b PROMOTES resolution; loss of FGF10 PROMOTES fibrosis
  - Therapeutic logic: find more "FGF10-like" agonizable receptors on epithelium

** THE CONCEPTUAL INVERSION (READ THIS FIRST) **
--------------------------------------------------------------------------------
This pipeline reuses the architecture of the epicardial/MI pipeline, BUT the
biology runs in the OPPOSITE direction. Do not copy the cardiac gene lists or
the perturbation interpretation verbatim.

  CARDIAC PIPELINE (epicardium):
    - Proliferation + EMT of epicardium = THERAPEUTIC GOAL (repair)
    - FGF10 = ACTIVATOR of a beneficial program
    - "Quiescent -> activated" is the GOOD transition
    - Receptor whose DELETION blocks activation = target to AGONIZE

  PULMONARY FIBROSIS PIPELINE (alveolar epithelium):
    - Proliferation + partial-EMT + transitional arrest = THE DISEASE
    - FGF10 = PROTECTIVE / pro-regenerative factor
    - "Successful AT2 -> AT1 regeneration" is the GOOD transition
    - "AT2 -> arrested Krt8+/basaloid state" is the BAD transition
    - Receptor whose OVEREXPRESSION pushes cells toward regeneration = target
    - Equivalently: receptor whose DELETION pushes cells toward disease

  => The phenotype label, the geneset of interest, and the SIGN of the in silico
     perturbation readout all flip relative to the cardiac pipeline.

SENDER / RECEIVER FRAMEWORK:
  - Sender cells (FGF10 / niche signal source):
      Alveolar (lipo)fibroblasts, airway smooth muscle cells, myofibroblasts,
      profibrotic macrophages
  - Receiver cells (your target population):
      AT2 cells and injury-activated alveolar progenitors (FGFR2b+)
  - Goal: find receptors on epithelium that, when engaged, bias cells toward
    completed regeneration rather than transitional arrest

POSITIVE CONTROL: FGF10 -> FGFR2 (FGFR2b isoform). Must rank highly throughout.
  In silico Fgf10/Fgfr2b deletion should shift cells toward the diseased state
  (this is the in silico equivalent of the Fgf10-deficient mouse phenotype).

OPTIONAL PARALLEL ARM (B) — ANTAGONIST DISCOVERY:
  Run the same machinery with senders/receivers configured to find drivers of the
  PATHOLOGICAL program (e.g., fibroblast -> myofibroblast; epithelium -> basaloid)
  that you would want to BLOCK. Positive controls there: TGFBR1/2, AXL, etc.
  Build Arm (A) first; (B) reuses the same code with different labels/genesets.

2.3 STANDARD scRNA-seq PROCESSING (per dataset)
--------------------------------------------------------------------------------
```R
library(Seurat)
library(harmony)

# ---- generic QC (note: mouse mito = "^mt-", human mito = "^MT-") ----
process_one <- function(counts, mito_pattern, batch_key = NULL) {
  obj <- CreateSeuratObject(counts = counts)
  obj[["percent.mt"]] <- PercentageFeatureSet(obj, pattern = mito_pattern)
  # Epithelium is fragile; keep filters permissive enough to retain AT2/AT1
  obj <- subset(obj, subset = nFeature_RNA > 200 &
                              nFeature_RNA < 7000 &
                              percent.mt < 20)
  obj <- SCTransform(obj, verbose = FALSE)
  obj <- RunPCA(obj)
  if (!is.null(batch_key)) obj <- RunHarmony(obj, group.by.vars = batch_key)
  red <- if (!is.null(batch_key)) "harmony" else "pca"
  obj <- RunUMAP(obj, reduction = red, dims = 1:30)
  obj <- FindNeighbors(obj, reduction = red, dims = 1:30)
  obj <- FindClusters(obj, resolution = 0.5)
  obj
}
```

NOTE ON EPITHELIAL UNDERSAMPLING: whole-lung dissociation underrepresents and
stresses epithelium. Where the original study provides EpCAM+/sorted epithelial
fractions (or use Kobayashi/Choi), prefer those for resolving the RECEIVER
trajectory. Whole-lung Strunz data is excellent for SENDER characterization.

The core difference from the cardiac pipeline: define TWO opposing trajectories
and contrast them. The "good" axis is the agonism target; the "bad" axis is what
you are trying to avoid.

3.1 REGENERATION-SUCCESS SIGNATURE (the GOOD state — to promote)

```R
                          name = "Transitional")
```

** WARNING — THE TRAP FROM THE CARDIAC PIPELINE **
A naive proliferation/EMT module score will mislabel disease-arrested,
partial-EMT, senescent transitional cells as "activated/good." Define the target
phenotype by TRAJECTORY COMPLETION (reaching AT1), not by EMT/proliferation
magnitude. Proliferation and EMT scores are CONFOUNDERS here, not the goal.

3.3 TEMPORAL / TRAJECTORY LABELING (mouse time course)
--------------------------------------------------------------------------------
```R
# Use Strunz time course + pseudotime (slingshot/monocle3) to order:
#   AT2  ->  Krt8+ transitional  ->  AT1
# Define two endpoints of the bifurcation:
#   COMPLETING = transitional cells progressing to AT1 (RegenSuccess up over time)
#   ARRESTED   = transitional cells persisting (Transitional/senescence stays high)
# Cells from late timepoints (d21/d28) still Krt8+ = arrested/disease-like.
epi_obj$fate_label <- dplyr::case_when(
  epi_obj$RegenSuccess1 > q_hi & epi_obj$Transitional1 < q_lo ~ "completing",
  epi_obj$Transitional1 > q_hi & epi_obj$RegenSuccess1 < q_lo ~ "arrested",
  TRUE ~ "intermediate"
)
```

```R
  filter(from %in% expressed_sender & to %in% expressed_receiver) %>%
```

5.2 FINE-TUNE: completing vs arrested
--------------------------------------------------------------------------------
```python
from transformers import BertForSequenceClassification, Trainer, TrainingArguments
from datasets import load_from_disk
from sklearn.metrics import accuracy_score, f1_score
import numpy as np

ds = load_from_disk("./tokenized/lung_epi.dataset").train_test_split(0.2, seed=42)
model = BertForSequenceClassification.from_pretrained(
    "ctheodoris/Geneformer", num_labels=2)

def metrics(p):
    preds = np.argmax(p.predictions, axis=1)
    return {"accuracy": accuracy_score(p.label_ids, preds),
            "f1": f1_score(p.label_ids, preds)}

args = TrainingArguments(output_dir="./gf_lung/", num_train_epochs=5,
    per_device_train_batch_size=8, per_device_eval_batch_size=8,
    warmup_steps=500, weight_decay=0.01, evaluation_strategy="epoch",
    save_strategy="epoch", load_best_model_at_end=True,
    metric_for_best_model="f1", logging_steps=100)

Trainer(model=model, args=args, train_dataset=ds["train"],
        eval_dataset=ds["test"], compute_metrics=metrics).train()
```

5.3 IN SILICO PERTURBATION (note the SIGN FLIP)
--------------------------------------------------------------------------------
```python
from geneformer import InSilicoPerturber

candidate_receptors = [
    "FGFR2","FGFR1",           # FGF axis (positive control; FGFR2b)
    "EGFR","ERBB2","ERBB3","MET","IGF1R","AXL",
    "BMPR1A","BMPR2","ACVRL1", # BMP (often pro-regenerative in alveolus)
    "TGFBR1","TGFBR2",         # pro-fibrotic (expect OPPOSITE direction)
    "PDGFRA","PDGFRB","NOTCH1","NOTCH2","FZD1","FZD2"
]

# OVEREXPRESS to find receptors that PUSH cells toward "completing" (label 1)
perturber = InSilicoPerturber(perturb_type="overexpress",
    model_type="CellClassifier", num_classes=2, emb_mode="cell",
    filter_data={"cell_type": ["AT2","transitional"]})
perturber.perturb_data("./gf_lung/", "./tokenized/lung_epi.dataset",
    "./results/geneformer/oe/", genes_to_perturb=candidate_receptors)

# Also DELETE to confirm the disease-driving direction
perturber_del = InSilicoPerturber(perturb_type="delete",
    model_type="CellClassifier", num_classes=2, emb_mode="cell",
    filter_data={"cell_type": ["AT2","transitional"]})
perturber_del.perturb_data("./gf_lung/", "./tokenized/lung_epi.dataset",
    "./results/geneformer/del/", genes_to_perturb=candidate_receptors)
```

5.4 RANK + POSITIVE-CONTROL VALIDATION
--------------------------------------------------------------------------------
```python
import pandas as pd
oe  = pd.read_csv("./results/geneformer/oe/perturbation_stats.csv")
dele = pd.read_csv("./results/geneformer/del/perturbation_stats.csv")

# OVEREXPRESSION effect: shift toward "completing" (class 1) = good agonist target
oe["regen_gain"] = oe["prob_class1_perturbed"] - oe["prob_class1_original"]
oe_sorted = oe.sort_values("regen_gain", ascending=False)

# DELETION effect: shift toward "arrested" (class 0) = required for regeneration
dele["regen_loss"] = dele["prob_class1_perturbed"] - dele["prob_class1_original"]

# *** POSITIVE CONTROL ***  (your Fgf10-KO mouse, in silico)
#   FGFR2 OVEREXPRESSION  -> strong positive regen_gain (rescue)
#   FGFR2 DELETION        -> strong negative (pushes to disease) == the KO phenotype
print("FGFR2 overexpression gain:", oe[oe.gene=="FGFR2"]["regen_gain"].values)
print("FGFR2 deletion loss:",       dele[dele.gene=="FGFR2"]["regen_loss"].values)

# Top novel agonist candidates = high regen_gain on OE, negative on deletion,
# EXCLUDING the known positive controls.
print(oe_sorted.head(15)[["gene","regen_gain"]])
```

INTERPRETATION:
  - Best targets: OE increases "completing" probability AND deletion decreases it
    (consistent causal direction), surface-expressed on AT2, druggable.
  - TGFBR1/2 should show the OPPOSITE pattern (OE -> arrest) — a useful internal
    negative-direction control and the seed of Arm (B).
